# Golang Bios Unlock Tool V1


## Run Locally

Clone the project

```bash
  git clone https://github.com/ZachC137/GolangBiosUnlock.git
```

Go to the project directory

```bash
  cd GolangBiosUnlock
```

Install dependencies

```bash
  go get
```

Start the server

```bash
  go build -ldflags="-w -s -H=windowsgui" -o GolangBiosUnlock.exe
```
